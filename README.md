# Triangular-Signal-Generator

## Overview

This project presents the design and simulation of a transistor-based triangular signal generator developed as part of an Analog Electronics course.

The circuit generates a triangular waveform with adjustable frequency and amplitude while operating from a single power supply. The design includes multiple analog building blocks such as a differential amplifier, current mirror, voltage amplifier, square-wave oscillator, triangular-wave generator, Class B output stage and negative feedback network.

The circuit was designed to operate over a frequency range of **0.2 kHz to 4 kHz**, with an adjustable output amplitude between **0.2 V and 1 V**, while driving a **2 kΩ load**.

Simulation results were used to validate the circuit performance under different operating conditions, including frequency, amplitude and temperature variations.

The project demonstrates the complete design process of an analog electronic circuit, from schematic design and component selection to simulation and performance evaluation.

## Features

- Transistor-based analog circuit
- Adjustable output frequency (0.2–4 kHz)
- Adjustable output amplitude (0.2–1 V)
- Single supply operation
- Differential amplifier input stage
- Current mirror biasing
- Class B output stage
- Negative feedback stabilization
- Analog circuit simulation
